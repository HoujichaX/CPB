# Contributing to CPB

Thank you for your interest in contributing to CPB!

## How to Contribute

1. **Report bugs** — Open an issue with reproduction steps
2. **Suggest features** — Open an issue with your use case
3. **Submit code** — Fork, branch, commit, pull request

## Development Setup

1. Install Visual Studio 2019+ with C++ workload
2. Open Developer Command Prompt (x64)
3. Run `build.bat`

## Code Style

- C++17
- 4-space indentation
- `snake_case` for functions and variables
- `PascalCase` for types and classes
- Comments in English (Japanese OK in documentation)

## Testing

Before submitting a PR, ensure all existing tests pass.
