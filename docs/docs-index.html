What is CPB?

A multi-layer archive format that compresses, protects, and disguises your data.

📖

Structure Dictionary

L3 Genre DSL detects data type and replaces recurring patterns with compact references.

🔤

String Dictionary

L5 GenDict caches exact matches. Second encounter compresses to ~279 bytes.

🛡️

Error Recovery

Reed-Solomon codes detect and repair corruption automatically. No re-download needed.

🎬

Video Container

Data stored as standard AVI. Compatible with every CDN, player, and forensic tool.

🔀

4D–16D Shuffle

Multi-dimensional byte rearrangement across up to 16 independent axes.

🔍

Full-Text Search

FIDX index enables keyword search inside archives without extraction.

Pipeline

Data flows through up to 6 layers. Each layer is independently toggleable.
